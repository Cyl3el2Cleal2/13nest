# NestJS App

A NestJS TypeScript application.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

The application will be available at:
- http://localhost:3000
- API Documentation: http://localhost:3000/api-docs

## Testing

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
